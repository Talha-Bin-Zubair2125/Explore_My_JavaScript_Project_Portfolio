What is the difference between == and === in JavaScript?

== checks for value equality, but allows type coercion (e.g., "5" == 5 is true).

=== checks for strict equality (value and type), so "5" === 5 is false.

What does async/await do in JavaScript?

async makes a function return a Promise.

await pauses the execution inside an async function until the Promise resolves, making async code look synchronous 


                async function fetchData() {
                    let result = await fetch("https://api.example.com");
                    let data = await result.json();
                    console.log(data);
                }
What is NaN in JavaScript, and when do we get it?

NaN stands for "Not a Number".It occurs when an operation fails to return a valid number.

let result = "hello" - 5;  // NaN
